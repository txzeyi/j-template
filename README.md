## 项目结构

    ```
    yj-admin
    ├── config                     // 配置相关
    ├── src                        // 源代码
    │   ├── api                    // 所有请求
    │   ├── assets                 // 主题 字体等静态资源
    │   ├── components             // 全局公用组件
    |   |   ├── business           // 基础组件
    |   |   ├── common             // 公共组件
    |   |   ├── costom             // 业务组件
    │   ├── directive              // 全局指令
    |   ├── enum                   // 枚举

    ```
